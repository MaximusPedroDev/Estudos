
### O elemento html 

criado pelas tags hmtl é criado um elemento html no qual o navegador pega o conteúdo dentro da delimitação das tags e define o que cada um seria 

um **==elemento==** html ou web é definido por 
```
<nome da tag> O conteúdo vai aqui... </nome de tag>
```

> [!**Nota:** Alguns elementos HTML não têm conteúdo (como o <br> elemento). Esses elementos são chamados de elementos vazios. Elementos vazios não têm tag final!]
