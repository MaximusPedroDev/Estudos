### Definição

O Html é uma linguagem de marcação de texto ( *Linguagem de Marcação de Hipertexto*)
é a linguagem padrão de marcação para criar paginas da web e responsável por descrever a estrutura da pagina por uma serie de elementos chamado de [Tag Html](htmlTags.md) onde cada uma faz como uma marcação para o navegador entender qual parte do código seria responsável pelo o que
Exemplo

```
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
```

#### Exemplo explicado

- A declaração define que este documento é um documento HTML5`<!DOCTYPE html>`
- O elemento é o elemento raiz de um HTML Página`<html>`
- O elemento contém metainformações sobre o Página HTML`<head>`
- O elemento especifica um título para o Página HTML (que é mostrada na barra de título do navegador ou na aba da página)`<title>`
- O elemento define o e é um contêiner para todos os conteúdos visíveis, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.`<body>`
- O elemento define um título grande`<h1>`
- O elemento define um parágrafo`<p>`

### Navegadores Web

O proposito de um navegador web é entender os códigos html e aparesentar na tela de forma correta .
o navegador não exibe a tag nem seu fechamento apenas o seu conteúdo modificado da forma que a tag representa
