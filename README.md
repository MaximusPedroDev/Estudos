# Repositorio de estudos de programação

Esse repositorio tem a finalidade de ter todas as aulas e materiais de meus estudos e pesquisas na area de programação

Todas as pastas deveram ser configuradas com a pasta da aula seu arquivo da linguagem estudada e um arquivo markdown com o assunto estudado a fim de manter uma legibilidade e organização otimizada ao revisar o conteudo

# Ordem das pastas desse repositorio

1. Configuração do repositorio ( como os arquivos vão ser organizados )
2. estudos pessoais (repositorio para estudos soltos como videos e pesquisas soltas)
   2.1 videos (como do youtube ou biblioteca da rocketseat entre outros )
   2.2 pesquisas ( sites com materias e assuntos soltos que possam ser importantes para o conhecimento)
3. cheats e comandos (um repositorio separado por tecnologia com os comandos da linguagem existentes para pesquisa rapida só do codigo , não precisa necessariamente de arquivo markdown)
4. cursos listados por linguagens , modulos de cada curso e suas aulas . tendo arquivo markdown para os assuntos
